Привязывание групп безопасности

Обсуждавшиеся ранее группы безопасности, как правило, требуются для обеспечения сетевого обмена с экземпляром всякий раз, когда группа безопасности по умолчанию для проекта изменяется для снижения требований.

Добавление групп безопасности обычно выполняется при загрузке экземпляра. При запуске с инструментальной панели, вы делаете это с вкладки Access & Security (Доступ и Безопасность) в диалоге Launch Instance (Запуск экземпляра). При запуске из командной строки присоединяйте к --security-groups разделенный запятыми список групп безопасности.

Кроме того, можно добавлять и удалять группы безопасности при запущенном экземпляре. В настоящее время эта функция доступна только через инструменты командной строки. Вот пример:

$ nova add-secgroup <server> <securitygroup>
$ nova remove-secgroup <server> <securitygroup>
