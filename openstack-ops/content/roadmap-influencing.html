Влияние дорожной карты

OpenStack действительно приветствует ваши идеи (и ваш вклад) и высоко цеенит обратную связь с реальными пользователями программного обеспечения. Немного обучившись процессу, который определяет развитие функционала, вы можете принять участие и, возможно, получить желаемые вами дополнения.

Просьбы о новых возможностях, как правило, начинают свою жизнь в Etherpad, инструменте для совместного редактирования, который используется для принятия координирующих замечаний в сессиях по данной функциональности на саммите разработчиков. Это приводит затем к созданию детального плана на сайте Launchpad для определенного проекта, который используется для более формального описания функций. Детальные планы затем одобряются членами проектной команды и разработка может начинаться.

Таким образом, самый быстрый способ получения вашей функциональности запрашиваемой для рассмотрения является создание Etherpad с вашими идеями и предложением сессии на саммите разработчиков. Если саммит разработчиков уже прошел, вы также можете напрямую создать некий детальный план. Ознакомьтесь с этим блогом о том как работать с детальными планами проекцией Виктории Мартинез, разработчика стажера.

Дорожную карту для следующей редакции в ее развитии можно увидеть в Редакциях.

Для определения потенциальных возможностей, предполагаемых в будущих редакциях, или для просмотра функционала, реализованного ранее, взгляните на существующие детальные планы, такие как Детальные планы вычислительной среды OpenStack (nova), Детальные планы идентификации OpenStack (keystone), атакже замечания к редакции.

Помимо пути прямого следования детальному плану, существует другой очень хорошо рассмотренный механизм воздействия на дорожную карту: опрос пользователей. Находящийся по адресу http://openstack.org/user-survey, он позволяет вам предоставить детали вашего развертывания и потребностей по умолчанию анонимно. В каждом цикле комитет пользователей анализирует результаты и создает отчет, который содержит предоставление конкретной информации техническому комитету и техническим руководителям проектов.
